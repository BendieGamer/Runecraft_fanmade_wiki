RuneCraft Patch 0.6.1

Release Date: 11/13/2021

⚠️ The Patch Notes you are reading are no longer current. Ensure you are using the latest version of RuneCraft.
Next Patch Notes: 0.6.2 - 11/23/2021

Overview

RuneCraft 0.6.1 brings a slew of bug fixes, improvements, and balance changes to many parts of the add-on. Notably, this update patches an AFK agility XP farm and optimizes the Thieving skill system.

Patch Notes

General - Bug Fixes

  • Mastery Cloaks now give the correct effect values
  • Pigs now reward the correct combat XP
  • Treasure Goblins' spawn rules removed

General - Changes

  • All armor now plays an equip sound
  • Particles from armor effects have been removed
  • Level-up sounds now audible to players within 12 blocks (was 6)
  • Wither and Ender Dragon have higher Party Hat drop chance
  • Their armor sets have been removed
  • Realm loading screen removed

Agility

  • Fixed exploit for AFK agility farming
  • Graceful Cape durability: 55 → 165
  • Graceful Top: 80 → 240
  • Graceful Leggings: 75 → 225
  • Graceful Boots: 65 → 195

Prayer - Changes

  • Diamonds in Zamorak Temple chests slightly reduced
  • Party Hat drop chance increased
  • Temple spawn frequency: every 49 chunks (was 64)

Thieving - Bug Fixes

  • No XP from failed steals
  • No level-up during stealing
  • Piglins/Wandering Traders give correct XP
  • Piglins no longer make extra sounds
  • Item Frames can be stolen from Cartographer Villagers
  • Shepherds can be stolen from
  • General thieving improvements

Thieving - Changes

  • Rogue Mask durability: 55 → 165
  • Rogue Top: 80 → 240
  • Rogue Trousers: 75 → 225
  • Rogue Boots: 65 → 195
  • XP changes:
    • Fletchers: 560 → 550
    • Weaponsmiths: 630 → 610
    • Wandering Traders: 780 → 730
    • Armorsmiths: 860 → 790
    • Clerics: 920 → 850
    • Librarians: 1010 → 910
    • Piglins: 1100 → 970
